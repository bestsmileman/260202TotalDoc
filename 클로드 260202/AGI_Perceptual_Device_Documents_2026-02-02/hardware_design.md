## 4. 감정셀 설계

### 4.1 감정셀 개념

```
감정셀 = 감정 상태를 물리적으로 저장하고 표현하는 장치

특징:
- 실시간 감정 저장
- 색상 신호 생성
- 온도 신호 생성

```

### 4.2 하드웨어 구성

#### 4.2.1 색상 생성 모듈

**RGB LED 매트릭스**
```
타입: WS2812B 또는 APA102
개수: 64개 (8×8 매트릭스)
색상: 16,777,216색 (24-bit)
밝기: 256단계
배치: 표면 전체 분산

제어:
- PWM 방식
- 개별 LED 제어 가능
- 그라데이션 구현
```

**유기 EL (선택사항)**
```
타입: Flexible OLED
크기: 50mm × 30mm
해상도: 128×64 pixels
색상: Full Color
장점: 자연스러운 색 변화
```

#### 4.2.2 온도 생성 모듈

**열전 소자 (Peltier Element)**
```
타입: TEC1-12706
크기: 40mm × 40mm × 3.6mm
전압: 12V DC
전류: 6A (최대)
온도 범위: -10°C ~ +50°C (ΔT)

배치: 4개
- 얼굴 영역: 2개
- 손 영역: 2개

제어:
- PWM H-Bridge 드라이버
- 온도 센서 피드백 (NTC)
- PID 제어 알고리즘
```

**온도 센서**
```
타입: MLX90614 (비접촉)
정밀도: ±0.5°C
범위: -40°C ~ +125°C
응답: < 1초
```

#### 4.2.3 힘 생성 모듈

**형상기억합금 (SMA)**
```
타입: Nitinol Wire
직경: 0.15mm ~ 0.5mm
구동 온도: 70°C
수축률: 3-5%
배치: 표정 근육 모방

용도:
- 얼굴 표정 (미세 조정)
- 손가락 움직임
```

**모터 액추에이터**
```
타입: Servo Motor (MG90S)
개수: 12개
토크: 2.5kg·cm
속도: 0.1s/60°
제어: PWM

배치:
- 입 (상하): 2개
- 눈썹: 2개
- 눈: 4개
- 손가락: 4개
```

**힘 센서**
```
타입: FSR (Force Sensing Resistor)
범위: 0.1N ~ 10N
정밀도: ±5%
응답: < 5ms

배치:
- 손바닥: 2개
- 손가락 끝: 5개
```

### 5.4 펌웨어

```c
// 메인 루프 (의사코드)
void main_loop() {
    while (1) {
        // 1. 활동신호 읽기
        ActivitySignal activity = read_activity_signal();
        
        // 2. 기억 검색
        MemoryContext context = search_memory(activity);
        
        // 3. 감정 생성
        EmotionSignal emotion = generate_emotion(activity, context);
        
        // 4. 휴호니즘 필터
        emotion = apply_huhoism_filter(emotion);
        
        // 5. 3원 변환
        ExpressionSignal expression = convert_to_3d(emotion);
        
        // 6. 감정셀에 출력
        output_to_emotion_cell(expression);
        
        // 7. 기억셀에 저장
        save_to_memory_cell(activity, emotion, expression);
        
        // 100ms 주기
        delay(100);
    }
}
```
